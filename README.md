# Ruboty::Phraseapp

[Ruboty](http://github.com/r7kamura/ruboty) plug-in to control [Phraseapp](https://phraseapp.com/)

## Install

```ruby
#Gemfile
gem "ruboty-phraseapp"
```

## Prepare

Set PHRASEAPP_ACCESS_TOKEN and PHRASEAPP_PROJECT_ID.

## Usage

### translation stats

```
@ruboty phraseapp stats locale <locale_id>
```
